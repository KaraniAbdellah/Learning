What is Express Js & APIs:

-- API:
    Way for two computer talk to each others
    by sending request and get the reponse

-- REST API
    is a type of API that uses HTTP requests 
    (like GET, POST, PUT, DELETE, PATCH) to communicate 
    between clients and servers.

-- Note: 
    API is the general concept while REST API
    is a specific way of implementing an API

-- Express Js
    Express.js is a framework that makes it easier to 
    work with Node.js. It simplifies handling requests, 
    routing, and creating web APIs

-- Example: making a website that can display the details of books,
    instead of make for each book details i can get the API 
    that contain ALL Details.
